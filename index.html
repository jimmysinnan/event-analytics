Sécurité & Propriété des données

Vos données restent sous votre contrôle.

Selon vos contraintes, Event Analytics peut être déployé en local, dans un environnement dédié ou dans une version hébergée sécurisée. Vos données ne vous appartiennent qu'à vous.

Déploiement local

Sur votre machine ou vos serveurs internes. Données 100% locales, aucun accès réseau requis. Vous gérez les sauvegardes.

Environnement dédié

Infrastructure dédiée, isolée, gérée conjointement. Accès multi-utilisateurs, sauvegardes encadrées, maintenance définie contractuellement.

Version hébergée sécurisée

Hébergement sécurisé avec accès contrôlé. Vos données restent isolées — aucune mutualisation, aucune revente, aucun usage tiers.

Vous êtes propriétaire de vos données

Vos données événementielles, billetterie, client et financières ne sont jamais revendues, ni partagées, ni utilisées à d'autres fins.

Conformité RGPD facilitée

Emails, profils d'achat, données nominatives traités dans votre périmètre. Pas de sous-traitant tiers à déclarer pour le traitement courant.

Accès aux données encadré contractuellement

En cas d'intervention support ou prestation, l'accès à vos données est limité au périmètre validé. Aucune action non sollicitée.

Sauvegardes selon votre mode de déploiement

Local : vous gérez vos archives. Environnement dédié ou hébergé : sauvegardes définies dans le contrat de service.
